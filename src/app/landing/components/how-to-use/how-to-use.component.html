How to Use

Step 1

Upload a translation file in .xlf format.

It's completely safe, since we do everything locally and your translations don't leave the browser.

Step 2

Add/Edit the missing translation keys

Start translating using our editor, customized to boost your performance.

Step 3

Download the translated file

Once you're done, just click "Download" and paste the ready-to-use generated file in your application.
